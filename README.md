# CV Matcher

### 🚀 Deployment

Deployed on Vercel

### ⚡ How to run this project

```bash
# Clone this project
git clone https://github.com/cvez-team/frontend.git

# Change directory to the project
cd ./frontend

# Install node dependencies
npm i

# Start development
npm run dev
```
